Objects Module

Powerful object detection and classification capabilities

Overview

The Objects module provides a high-level interface for object detection and classification using YOLOv8. It supports multiple detection strategies, custom models, and real-time processing.

Usage

from yolozone import Objects

# Initialize detector
detector = Objects()

# Detect objects in an image
results = detector.detect('image.jpg')

# Process video stream
detector.process_video('video.mp4', output='output.mp4')

Methods

detect(source)

Detect objects in an image or video frame.

Parameters

  • source (str | np.ndarray): Path to image or numpy array

Returns

  • List[Detection]: List of detected objects with coordinates and confidence scores

process_video(source, output=None)

Process video file or stream and detect objects in each frame.

Parameters

  • source (str): Path to video file or stream URL
  • output (str, optional): Path to save output video

set_model(model_path)

Load a custom YOLOv8 model for detection.

Parameters

  • model_path (str): Path to custom model weights

Examples

Basic Detection

from yolozone import Objects

detector = Objects()
results = detector.detect('image.jpg')

for obj in results:
    print(f"Found {obj.class_name} with confidence {obj.confidence}")

Custom Model

detector = Objects()
detector.set_model('path/to/custom/model.pt')
results = detector.detect('image.jpg')

Video Processing

detector = Objects()
detector.process_video(
    'input.mp4',
    output='output.mp4'
)
