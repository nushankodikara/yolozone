Tracker Module

Robust object tracking and motion analysis

Overview

The Tracker module provides advanced object tracking capabilities, enabling you to track objects across video frames, analyze their motion patterns, and generate trajectory data.

Usage

from yolozone import Tracker

# Initialize tracker
tracker = Tracker()

# Track objects in video
tracks = tracker.track_video('video.mp4')

# Get trajectory data
trajectories = tracker.get_trajectories(tracks)

Methods

track_video(source, output=None)

Track objects throughout a video sequence.

Parameters

  • source (str): Path to video file or stream URL
  • output (str, optional): Path to save annotated video

Returns

  • List[Track]: List of tracked objects with their trajectories

get_trajectories(tracks)

Extract trajectory data from tracked objects.

Parameters

  • tracks (List[Track]): List of tracked objects

Returns

  • Dict[int, List[Point]]: Dictionary mapping track IDs to trajectory points

analyze_motion(track)

Analyze motion patterns of a tracked object.

Parameters

  • track (Track): A tracked object instance

Returns

  • MotionAnalysis: Motion analysis results including speed, direction, etc.

Examples

Basic Tracking

from yolozone import Tracker

tracker = Tracker()
tracks = tracker.track_video(
    'traffic.mp4',
    output='tracked.mp4'
)

print(f"Tracked {len(tracks)} objects")

Trajectory Analysis

tracks = tracker.track_video('video.mp4')
trajectories = tracker.get_trajectories(tracks)

for track_id, points in trajectories.items():
    print(f"Track {track_id} has {len(points)} points")
    
    # Calculate total distance
    distance = sum(
        p1.distance(p2)
        for p1, p2 in zip(points, points[1:])
    )

Motion Analysis

tracks = tracker.track_video('video.mp4')

for track in tracks:
    motion = tracker.analyze_motion(track)
    
    print(f"Track {track.id}:")
    print(f"  Average speed: {motion.avg_speed}")
    print(f"  Direction: {motion.direction}")
    print(f"  Motion pattern: {motion.pattern}")
